# FaceCraft

An Android app for AI-powered 3D face morphing, built for the Nexa Ã— Qualcomm On-Device AI Hackathon.

## Features

- 3D face visualization with Three.js
- Natural language face modification prompts
- On-device VLM inference using NexaSDK on Qualcomm Hexagon NPU
- Real-time morph parameter adjustments

## Tech Stack

- Kotlin + Jetpack Compose
- NexaSDK with OmniNeural-4B-mobile VLM
- Three.js for 3D rendering
- Qualcomm Hexagon NPU acceleration
